### p5 instance mode

ブラウザ上のエディタで、instance mode ができたぽい

``` js
const s = (p) => {
  p.setup = () => {
    p.createCanvas(60, 40);
  }
  p.draw = () => {
    p.background(p.random() * 255);
  }
}

const myp5 = new p5(s);

```
